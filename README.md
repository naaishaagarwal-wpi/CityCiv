# CityCiv

This web app is meant to give teens and young adults a platform through which they can interact with, and learn more about, their local government and community. 

After downloading the CityCiv code, please open the powershell terminal in VS code, and type "npm run dev" and press enter. Then open the web app (http://localhost:3000/home.html), and the app will be displayed in front of you with full functionality.

For the first time running the code, type npm install. This will create a node_modules folder using the package.json file which will allow you to run npm run dev every time after.

We are using VS Code to run the code.

Once you open the app, you will enter our login/sign up page. You should enter a valid email and password and then you will be prompted to go to your email to verify your email (a supabase email will be sent to you). From here you will need to log in again and then you can access all the pages.

The MVP page is the Dashboard and the rest are part of our extensions. You can create a post with the add post button and when you create a post, the post will be added to the database on Supabase. You can also like/unlike and reply to posts. 

You can also view our app live hosted at this URL: https://city-civ-nvbz.vercel.app/
