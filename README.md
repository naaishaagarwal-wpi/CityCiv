# CityCiv

This web app is meant to give teens and young adults a platform through which they can interact with, and learn more about, their local government and community. 

After downloading the CityCiv code, please open the powershell terminal in VS code, and type "npm run dev" and press enter. Then open the web app (http://localhost:3000/home.html), and the app will be displayed in front of you with full functionality.
